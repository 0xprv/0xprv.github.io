## 💼 Startup / Freelance Work

### Realtime Monitoring of Bank Transactions using AI/ML in Nigeria

**Duration:** Jul 2024 – Apr 2025  
**Tech Stack:** Java (Spring Boot) · Rust · TypeScript/JavaScript · Python

---

### 🚀 Overview

This project was developed under an NDA with a Nigeria-based company.

I led the design and development of an **enterprise-grade transaction monitoring system** that integrates directly with banking infrastructure to process and analyze live financial transactions.

---

### ⚙️ System Functionality

- Real-time ingestion of bank transaction streams  
- Detection pipeline using:
  - Machine learning models  
  - Clustering techniques  
  - Rule-based anomaly detection  

- Automatic flagging of suspicious transactions based on predefined risk conditions  

---

### 🧠 Detailed Project Description

The system connects directly to banking servers and continuously monitors transaction flows in real time.

Each transaction passes through a multi-layer analysis pipeline:

1. **Preprocessing Layer**
   - Data normalization  
   - Feature extraction  
   - Noise filtering  

2. **Detection Layer**
   - ML models for anomaly detection  
   - Clustering to identify unusual behavior patterns  
   - Deterministic rules for known fraud signatures  

3. **Decision Engine**
   - Combines ML confidence scores with rule-based triggers  
   - Assigns risk levels to each transaction  

---

### 📦 Case Management System

- Suspicious transactions are grouped into **cases**
- Cases are automatically:
  - Assigned to bank personnel  
  - Routed based on branch mapping  

- Notification system:
  - Email-based alerts  
  - Actionable case summaries  

This ensures **fast human intervention and audit tracking**.

---

### 🏗️ Architecture Highlights

- High-throughput processing pipeline  
- Low-latency event handling  
- Hybrid architecture:
  - **Spring Boot** → orchestration & APIs  
  - **Rust** → high-performance processing  
  - **Python** → ML models  

- Modular design for:
  - Model upgrades  
  - Rule updates  
  - Scaling across banks
